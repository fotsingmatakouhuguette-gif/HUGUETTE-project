# AFRIPLAY
##  Project Overview
** AFRIPLAY** Is a gaming plateform digitizing traditional Cameroonian games like Riddles,marbles,and Nyama Nyama to preserve cultural heritage for digitl generation made with **Python (Pygame)**.  


**IN RIDDLES**, you jump,and answer the different riddles using  the 1,2,3,4 buttons

**MARBLES** you can game with your friend. you drag the blue ball backwards to help escape the balls in the square.

**NYAMA NYAMA** you press "Y" and "N" to answer "yes" or "no" to the different questions

This project was created as part of our **end of Fall 2025 Exams**.

##  Features
- Built with **Python & Pygame**  
- Dynamic **Riddle question generator**  
- **Multiple-choice** for the riddles and Nyama Nyama 
- Real-time **collision and obstacle mechanics**  
- **Keyboard controls** for movement and answer selection  
- Simple yet fun **educational  gameplay**
